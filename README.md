## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
## API Request
| Endpoint | HTTP Method | Description |
| ------ | ------ | ------ |
| /api/healthcheck	| GET |	Check if server is working |
| /api/balance/:address |	GET	| Get address balance |
